# **App Name**: RoutineFlow

## Core Features:

- Routine Input: Capture user's daily routines with details like attempt, time, work/subject, and score.
- Routine Display: Show daily routines in a structured table with attempt, time, work, score, and status.
- Status Indicator: Visually indicate the status of each routine based on the score using color-coding.
- Smart Scheduling Tool: AI powered tool suggests optimal times for each activity, using an analysis of past scores and routine data.
- Local Storage Persistence: Save routines to local storage to remember across browser sessions.

## Style Guidelines:

- Primary color: Deep sky blue (#4169E1), suggesting a sense of focus and reliability, against a light background.
- Background color: Very light blue (#F0F8FF), close to white but visually distinct to give a sense of spaciousness and organization.
- Accent color: Slate blue (#708090) for secondary actions and elements, complementing the primary color with a calm, understated tone.
- Body and headline font: 'PT Sans' sans-serif, offering a modern and slightly warm feel.
- Minimal, clear icons to represent different types of activities and subjects.
- Clean, table-based layout to display routines, with a clear separation of columns and rows for easy readability.
- Subtle transitions when toggling the form or updating the table to provide a smooth user experience.
